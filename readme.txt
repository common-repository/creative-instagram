Display beautifully clean, customizable, and responsive Instagram posts from multiple accounts. 

== Description ==

Social Portfolio Feed use to display instagram post in your website. Also, you can display your instagram posts in widget, page and post. It is very flexible and compatible on all devices like ipad,iphone smart phone etc. 


If you have any question contact us at here: [Plugin Support Forum ](http://wordpress.org/support/plugin/creative-instagram).

== Installation ==

1. Upload the entire ** social-portfolio-feed ** folder to the **/wp-content/plugins/** directory.
2. Activate the plugin through the **Plugins** menu in WordPress admin.
3. Create new gallery, Use **[CI id=320]** shortcode to publish your gallery into any page or post.
